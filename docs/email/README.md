# HTML-письма iStockLink

## product-update.html

Письмо для действующих пользователей о новых возможностях сервиса.

### Тема письма (варианты)

- **iStockLink обновился — воронка, поставщики и разбор КП**
- **Новое в iStockLink: короткий обзор в видео**
- **Ваш тендерный отдел в одном окне — что изменилось**

### Перед отправкой

1. Откройте `product-update.html` и замените плейсхолдеры:
   - `{{RUTUBE_VIDEO_ID}}` — ID ролика (32 символа из URL `https://rutube.ru/video/XXXXXXXX/`)
   - `{{SUPPORT_EMAIL}}` — email поддержки
   - `{{UNSUBSCRIBE_URL}}` — ссылка отписки (или удалите строку, если не нужна)

2. При необходимости отредактируйте блоки «Что появилось» под фактический релиз.

3. Отправьте **тестовое письмо себе** и проверьте в Gmail, Outlook и на телефоне.

### Превью Rutube в письме

Встроенный плеер в email не работает (блокируется почтовыми клиентами). Используется **кликабельное превью**:

- картинка: `https://i.rtimg.ru/vi/{VIDEO_ID}/xl/`
- ссылка: `https://rutube.ru/video/{VIDEO_ID}/`

Если превью не загружается — проверьте ID ролика. Альтернатива: скачайте обложку из Rutube и загрузите в сервис рассылки как обычное изображение.

### Как отправить

- **Unisender / SendPulse / Mailchimp** — вставьте HTML в редактор «свой HTML» / «код письма».
- **Из приложения** — если есть SMTP/API, передайте HTML-тело как `text/html`.
- **Вручную** — скопируйте в Thunderbird / Outlook (режим HTML), но тестируйте отображение.

### Локальный просмотр

Откройте файл в браузере (двойной клик или `file://`). Для проверки с подставленным ID:

```bash
sed 's/{{RUTUBE_VIDEO_ID}}/ВАШ_ID/g; s/{{SUPPORT_EMAIL}}/support@example.com/g; s|{{UNSUBSCRIBE_URL}}|#|g' \
  docs/email/product-update.html > /tmp/preview.html && xdg-open /tmp/preview.html
```
