CTA в Zero Block — iStockLink

Один Zero Block: синяя карточка + нативная форма Tilda. Заявки идут в раздел «Заявки». Открыть превью

Шаг 1. Создайте Zero Block

  1. На странице: + → Zero Block (или «Нулевой блок»).
  2. Высота артборда 1200px: 480px (десктоп).
  3. Высота артборда 320px: 520px (мобильный — подгоните визуально).
  4. Настройки блока → Якорь / Block anchor ID: demo (для ссылок #demo с тарифов).

Шаг 2. Элементы (снизу вверх по слоям)

Добавляйте через + в левом верхнем углу редактора Zero Block.

Шаг 3. CSS class у каждого элемента

Клик по элементу → Settings → поле CSS class name → вставьте класс из таблицы (без точки).

Шаг 4. Вставьте CSS в Zero Block

Настройки Zero Block → вкладка CSS → скопируйте весь код из файла zero-cta.css (без тегов <style>).

CSS 
/* см. файл zero-cta.css в этой папке — скопируйте целиком */

Шаг 5. Приём заявок

  1. Настройки сайта → Формы → Email (или Telegram) — подключите сервис.
  2. Клик по элементу Form → Settings → Services → галочка на Email.
  3. Опубликовать страницу.
  4. Отправить тест → проверить Заявки.
Блок формы не скрывать. В Zero Block форма — это элемент внутри блока. Прячутся только заголовки формы через CSS, поле и кнопка остаются видимыми.

Шаг 6. Кнопки тарифов

В блоках тарифов ссылка: #demo — скролл к этому Zero Block.

Если форма выглядит «не так»

  • Проверьте класс tg-zb-cta-form на элементе Form.
  • В форме уберите заголовок и описание в Контенте.
  • На мобильном в Zero Block переключите Form на вертикальную в настройках элемента.
  • Сдвиньте Y координаты текста и формы на брейкпоинтах 640 и 320.
