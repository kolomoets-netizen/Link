## Настройка в Telegram Ads

1. [ads.telegram.org](https://ads.telegram.org) → Создать объявление
2. Формат: **Sponsored Message** или **Channel Post**
3. Загрузить PNG 1200×628
4. URL: `https://kolomoets-netizen.github.io/Link/` (или ваш домен после Tilda)
5. Таргетинг: Россия, интересы «бизнес», «закупки», каналы про госзакупки

## A/B тест

Запустите все 3 варианта с одинаковым бюджетом 500–1000 ₽/день на 3 дня. Сравните CTR — обычно вариант «свой агрегатор» (№2) лучше конвертит у тех, кто уже в тендерах.
